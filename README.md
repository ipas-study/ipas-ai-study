# iPAS AI 應用規劃師（初級）備考學習站

🌐 **網站**：https://ipas-study.github.io/ipas-ai-study/

---

## 功能概覽

| 分頁 | 內容 |
|---|---|
| 首頁 | 倒數計時、考試介紹、最新 AI 消息（每週自動更新） |
| 學習指引 | 官方指引全文（初級 2 科 + 中級 3 科）；螢光重點 + 白話導讀 |
| 考古題庫 | **463 題**（官方 433 + 自編 30）；即時對答；⭐ 收藏 / ❌ 錯題本 |
| 模擬考 | 隨機抽題、計時作答、自動評分；支援中斷後繼續 |
| 名詞解釋 | 384 個 AI 專有名詞白話解釋 |
| AI 工具 | 22 個可落地工具 + 128 支教學影片（每週更新） |
| AI 人才 | 數位發展部 AI 人才地圖 |
| Vicky 社群 | 影片牆 + LINE 社群 QR code |
| 使用說明 | 功能導覽、備考建議、版本更新紀錄 |

## v3.0 新功能（2026-06-13）

- **⭐ 錯題本 & 收藏**：答題後可收藏題目；答錯自動記錄，可切換「我的錯題」專項複習
- **🤖 AI 導師**：一鍵複製解題 Prompt，貼給 Claude 即可獲得詳細解析
- **🎯 模擬考進度保存**：中斷後重開可繼續作答，不怕意外關掉視窗
- **📋 使用說明分頁**：功能導覽 + 30 天備考建議 + Changelog
- 導覽列爆行修正（11 個按鈕不換行）

## 自動化架構

每週一早上自動執行：蒐集 AI 新聞 / 新工具 / 教學影片 → 歷史保留合併 → 推送 GitHub → 網站 1~2 分鐘內更新
